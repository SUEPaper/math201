元组

tuple 与列表类似，但元组元素不能更改

元组创建​

tup1 = ("SUEP",1951,1.13)
#当元组元素只有一个时，要加‘，’
tup2 = ("SUEP",)

元组使用​

通过索引​

tup1 = ("SUEP",1951,1.13)
print(tup1[1])
"""
输出：
1951
"""

元组的运算​

元组具有+++与∗*∗这两种运算，+++为元组相连，∗*∗为元组重复,与列表类似

tup1 = ("SUEP",1951,1.13)
tup2 = ("xsm",)
print(tup1 + tup2)
print(tup2 * 2)
"""
输出：
('SUEP', 1951, 1.13, 'xsm')
('xsm', 'xsm')
"""

元组的切片​

与列表一样，元组也具有切片的功能

#语法为tuple[a:b],返回tuple中索引为a到b的元素
tu11 = ["SUEP",1951,1.13,19.39]
print(tu11 [1:4])
print(tu11 [4:1])
print(tu11 [-4:-1])
#也可以list[a:]，返回tuple中索引为a到结束的元素
print(tu11 [2:])
"""
输出：
[1951, 1.13, 19.39]
[]
['SUEP', 1951, 1.13]
[1.13, 19.39]
"""

可以使用的函数​

tul1 = ["SUEP",1951,1.13]
#获得list的长度
print(len(tul1))
#获得最大最小值
tul2 = [1,1,4,5,1]
print(min(tul2))
print(max(tul2))
"""
输出：
3
1
5
"""
